About This Project

Built for students,
designed with care

Automated QR Data Collection System is a student registration management system designed to simplify collecting, managing, and exporting student data for college events, workshops, and seminars.

Our platform is completely free to use. No setup fees, no hidden charges — just a clean, fast, and reliable tool that works for any college event.

Whether you're organizing a tech fest, cultural event, or academic seminar — our system handles all the registration work so you can focus on the event.

Features

What this project includes

📋

Dynamic Registration Form

Admin can customize form fields, add/remove fields, and update event details anytime without touching code.

📱

QR Code Generator

Generate a QR code for the registration form, download it, and share it anywhere — print, project, or WhatsApp.

🔒

Secure Admin Panel

Protected login, live registration table, search, delete, and CSV export — all in one clean dashboard.

🗄️

MongoDB Database

Permanent data storage — registrations don't disappear when the server restarts. Data is always safe.

🌐

Fully Deployable

Ready to host on Render.com for free. One GitHub push and the website is live on the internet.

📊

CSV Export

Export all registrations as a CSV file anytime. Opens directly in Excel or Google Sheets.
